# chat-App
🚀 Realtime Chat Application

A full-stack realtime group chat application built using React, Node.js, Express, and Socket.io. This project enables multiple users to join a common room, send messages instantly, and see typing indicators in real time.

✨ Features
💬 Realtime messaging using WebSockets
👥 Group chat functionality (single room)
⌨️ Live typing indicator
🕒 Message timestamps
🎨 Clean chat UI (WhatsApp-like design)
🔄 Instant message updates without refresh
🛠️ Tech Stack
Frontend
  React.js
  Tailwind CSS
Backend
  Node.js
  Express.js
  Socket.io
